id: types
title: Type notations
---

|      Notation       |                      Meaning                      |
| ------------------- | ------------------------------------------------- |
| Word                |                                                   |
| [Bracketed]         | The type is within a list.                        |
| With question mark? | The type is optional.                             |
| &lt;Braced>         | The type is wrapped, e.g. async within a promise. |
