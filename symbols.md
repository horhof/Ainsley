# Symbol index

| Symbols |  Position   | Meaning              |
| :-----: | :---------: | -------------------- |
|   `?`   |   Postfix   | Boolean              |
|   `.`   |   Postfix   | Procedure            |
|   `!`   |   Postfix   | Block lambda         |
|   `:`   |   Prefix    | Parameter list       |
|   `=`   |   Prefix    | Return value         |
|   `,`   |    Infix    | Separates parameters |
|  `[ ]`  | Surrounding | Optional term        |
|  `{ }`  | Surrounding | Lambda               |
|  `< >`  | Surrounding | Asynchronous         |
