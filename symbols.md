# Symbol index

| Symbols |            Meaning            |
| :-----: | ----------------------------- |
|   `?`   | Ends boolean                  |
|   `.`   | Ends procedure                |
|   `!`   | Ends unary lambdas            |
|   `:`   | Starts parameter list         |
|   `,`   | Separates parameters          |
|   `=`   | Starts return value           |
|  `[ ]`  | Surrounds optional parameters |
|  `{ }`  | Surrounds lambda              |
|  `< >`  | Surrounds asynchronous data   |
